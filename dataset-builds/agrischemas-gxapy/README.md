# agrischemas-gxapy

This is a library of utilities to convert the [EBI Gene Expression Atlas data](https://www.ebi.ac.uk/gxa) to RDF, following the BioSchemas/AgriSchemas specifications. 

This is part of the [AgriSchemas data builds](../README.md).

At least for the time being, the project isn't published on PyPI, since it's only used within AgriSchemas. However, it can be imported via github URL, eg, in a  [requirements file for details](../agrischemas-gxa/requirements-local.txt), or [using Poetry](https://python-poetry.org/docs/dependency-specification/#git-dependencies).
